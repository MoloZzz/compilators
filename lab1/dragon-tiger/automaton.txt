1. Мова, яку приймає автомат:
L={x,y,z,ϵ}

2. Недетермінованість:
Автомат не є детермінованим,
оскільки існують стани з множинними переходами без конкретного символу (epsilon transitions),
що викливаэ неоднозначність у виборі наступного стану:
1) З стану 1 можна перейти до станів 2, 3 або 4 без зчитування символу.
2) З стану 2 можна перейти до стану 5 без зчитування символу.
 Також з кожного стану можуть бути переходи на символи 𝑥 , 𝑦 x,y і 𝑧 z, що ще більше ускладнює автомат.

3. Детермінізація:
Для детермінізації автомата, необхідно створити новий автомат,
 у якому кожен стан однозначно визначається поєднанням множини станів оригінального автомата. 
- Початковий стан: {1, 2, 3, 4}
   - Перехідна таблиця:
     | Вхідний символ | Стан {1, 2, 3, 4} | Стан {5} | Стан {6} | Стан {7} |
     |----------------|------------------|---------|---------|---------|
     | x              | {5}              |         |         |         |
     | y              | {6}              |         |         |         |
     | z              |                  | {7}     |         |         |
   - Переходи:
     - {1, 2, 3, 4} -> x -> {5}
     - {1, 2, 3, 4} -> y -> {6}
     - {5} -> z -> {7}
